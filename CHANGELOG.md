# Changelog
All notable changes to this package will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/en/1.0.0/)
and this project adheres to [Semantic Versioning](http://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2022-10-16
- Initial release.

## [1.1.0] - 2024-01-05
- Addition of a `Quick Create` Menu.
- Descriptive names for various operations.
- Additional Menu options: `Clear, Reset` - Fields.
- Ability to perform operations on existing folders.
- Ability to perform operations on Asset's root folder.

## [1.2.1] - 2024-03-01
- Menu path changed to: `Racer -> DirTool` .
- A `DirTool/` folder(to store `MenuData.asset` file), gets created after this package is installed.
